
# `md5sum`

```bash
~$ md5sum file.txt
8a495f5fdcd9a5e0407c64920e58cd65  file.txt
```

---
# `sha256sum`

> Uses 256 bit encryption

```bash
~$ sha256sum file.txt
09307f8dd3869abc7701cf98b7047396eb55184d9d938751947300c2770464b9  file.txt
```
