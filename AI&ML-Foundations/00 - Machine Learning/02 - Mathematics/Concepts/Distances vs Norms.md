
When we talk about the **Distance** between two points $\mathbf{p}$ and $\mathbf{q}$, the formula is:

$$d_1(\mathbf{p}, \mathbf{q}) = \sum |p_i - q_i|$$

Here, the subtraction is explicit because you are measuring the gap between two different locations.

However, a **[[Norms and Lp|Norm]]** measures the distance from a vector to the **Origin $(0,0)$**. If we set $\mathbf{q} = (0,0)$, the formula becomes:

$$\|\mathbf{p}\|_1 = |p_x - 0| + |p_y - 0| = |p_x| + |p_y|$$
